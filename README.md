# MesaNews

#### Descrição

É um aplicativo iOS onde o usuário poderá ler noticias e marcar suas notícias favoritas.

#### Tecnologias utilizadas

1. Swift
2. UIKit
3. Cocoapods
4. Alamofire
5. AlamofireImage
6. SwiftKeychainWrapper

#### Arquitetura

- Model-View-ViewModel (MVVM)
